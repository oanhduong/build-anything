SELF-02 + SELF-09 — draft the candidate file `${PENDING_DIR}/${candidate_id}.md` with EXACTLY this frontmatter and body (tag is ALWAYS `model-crutch <current-model>`, NEVER `architecture` — SELF-09 hard constraint; current model token: use `claude-sonnet-4-6` as the default model-crutch version string unless a `CLAUDE_MODEL` env var is set, in which case use that):

       error-match: <lowercased TARGET token or generic error fragment>
       evidence: <the verbatim trace line — TIMESTAMP TOOL TARGET EXIT_CODE>

       ## What happened
       Command via <TOOL> on <TARGET> exited <EXIT_CODE> (non-zero). Auto-distilled from trace evidence on <UTC timestamp from `date -u +%Y-%m-%dT%H:%M:%SZ`>.
